Eigen::MatrixXd ConstraintSolver::calculatePinvJacobianBySVD ( Eigen::JacobiSVD< Eigen::MatrixXd >  svd) const
protected

Base method for calculation of the pseudoinverse Jacobian by using SVD.

Parameters
svdThe singular value decomposition object of a Jacobian.
Returns
A pseudoinverse Jacobian

Calculates the pseudoinverse of a Jacobian by using SVD. Truncation is active always. Means if calculated (damped) singular value is < than EPS than truncate the singular value to 0.0. Damping is active in case of > than static const DAMPING_LIMT, else it is assumed that no damping is active.
